## Electron 新建右键菜单

Electron 中通过 `Menu` 和 `MenuItem` 模块创建右键菜单（上下文菜单）。

### 基本用法

```javascript
const { Menu, MenuItem, BrowserWindow, app } = require('electron')

app.whenReady().then(() => {
  const win = new BrowserWindow({ width: 800, height: 600 })

  // 创建右键菜单
  const contextMenu = Menu.buildFromTemplate([
    {
      label: '复制',
      role: 'copy'
    },
    {
      label: '粘贴',
      role: 'paste'
    },
    { type: 'separator' },  // 分割线
    {
      label: '自定义操作',
      click: () => {
        console.log('点击了自定义操作')
      }
    }
  ])

  // 监听 context-menu 事件触发弹出
  win.webContents.on('context-menu', (event, params) => {
    contextMenu.popup({ window: win })
  })
})
```

### 在渲染进程中触发（推荐方式）

**主进程** `main.js`：
```javascript
const { ipcMain, Menu, BrowserWindow } = require('electron')

ipcMain.on('show-context-menu', (event) => {
  const win = BrowserWindow.fromWebContents(event.sender)

  const menu = Menu.buildFromTemplate([
    { label: '菜单项 1', click: () => event.sender.send('menu-click', 'item1') },
    { label: '菜单项 2', click: () => event.sender.send('menu-click', 'item2') },
    { type: 'separator' },
    { label: '关闭', role: 'close' }
  ])

  menu.popup({ window: win })
})
```

**渲染进程** `renderer.js`：
```javascript
const { ipcRenderer } = require('electron')

// 监听右键事件
window.addEventListener('contextmenu', (e) => {
  e.preventDefault()
  ipcRenderer.send('show-context-menu')
})

// 接收菜单点击回调
ipcRenderer.on('menu-click', (event, item) => {
  console.log('点击了:', item)
})
```

### MenuItem 常用属性

| 属性 | 说明 |
|------|------|
| `label` | 菜单文字 |
| `type` | `normal` / `separator` / `checkbox` / `radio` / `submenu` |
| `role` | 内置角色，如 `copy` `paste` `undo` `redo` `quit` 等 |
| `click` | 点击回调函数 |
| `enabled` | 是否可点击 |
| `visible` | 是否显示 |
| `checked` | checkbox/radio 状态 |
| `submenu` | 子菜单（嵌套数组） |
| `accelerator` | 快捷键，如 `'CmdOrCtrl+C'` |

### 带子菜单示例

```javascript
const menu = Menu.buildFromTemplate([
  {
    label: '更多操作',
    submenu: [
      { label: '子菜单 A', click: () => {} },
      { label: '子菜单 B', click: () => {} }
    ]
  },
  {
    label: '开关选项',
    type: 'checkbox',
    checked: true,
    click: (menuItem) => {
      console.log('当前状态:', menuItem.checked)
    }
  }
])
```

### 根据点击位置动态生成菜单

```javascript
win.webContents.on('context-menu', (event, params) => {
  const { x, y, selectionText, isEditable } = params

  const menuTemplate = []

  if (selectionText) {
    menuTemplate.push({ label: `复制 "${selectionText}"`, role: 'copy' })
  }

  if (isEditable) {
    menuTemplate.push({ label: '粘贴', role: 'paste' })
  }

  menuTemplate.push({ type: 'separator' })
  menuTemplate.push({ label: `坐标: (${x}, ${y})`, enabled: false })

  Menu.buildFromTemplate(menuTemplate).popup({ window: win })
})
```

`context-menu` 事件的 `params` 包含丰富的上下文信息（点击坐标、是否可编辑、选中文字等），非常适合做动态菜单。
